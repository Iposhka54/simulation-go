Проект “Симуляция” на GO

Суть проекта — пошаговая симуляция 2D мира, населённого травоядными и хищниками.
Кроме существ, мир содержит ресурсы (траву), которыми питаются травоядные,
и статичные объекты, с которыми нельзя взаимодействовать — они просто занимают место.

2D мир представляет собой матрицу N×M, каждое существо или объект занимают клетку целиком,
нахождение в клетке нескольких объектов/существ — недопустимо.

Идея взята [отсюда](https://www.youtube.com/watch?v=SfEZSyvbj2w) и упрощена.

Мотивация проекта #
Основная цель — продемонстрировать архитектурный дизайн в Go, где:

вместо иерархии классов — композиция и интерфейсы,
вместо наследования — встраивание и интерфейсная типизация,
вместо “умных объектов” — “глупые данные” + “умные сервисы”.

Подробное описание проекта: [Go Simulation](https://zhukovsd.github.io/golang-backend-learning-course/projects/simulation/)
